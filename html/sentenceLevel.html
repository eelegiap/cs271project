Word Frequency
This visualization displays frequently used words in each text. The displayed words were chosen using the TF-IDF algorithm in order to find the most semantically relevant words. Hovering over each bar will display information about how many times the word is used, along with the number of sentences the word has appeared in.
The “Elements” selection box allows users to choose whether they would like to see 5, 10, or 15 elements displayed. The “Text” selection box allows users to choose whether they would like to see word frequencies for the source or translation document.
Sentence Length
This visualization displays the distribution of sentences based on their lengths. Sentence length is measured on the x-axis and frequency is measured on the y-axis.
Hovering over an individual bar will provide specific information on the sentence length corresponding to the selected bar and the frequency at which sentences of said length occur.
About the project
How were the alignments created?

The words and sentences in the original and translated texts were aligned computationally with manual corrections. The machine sentence alignment uses Bleualign, an algorithm for aligning parallel texts that compares Bleu similarity scores between the machine-translated original and the human-translated text. Once the sentences are aligned, we aligned words using SimAlign, an open-source tool that aligns words in parallel sentences using pretrained language models. Our alignments are based on mBERT, which is supported in 104 languages. We processed the texts into tokens using the popular Python library Spacy. Metrics seen in the graphs are based on different counting methods from corpus linguistics.
